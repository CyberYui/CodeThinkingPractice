# 09 Markdown转HTML

## 目标

把简单的Markdown文本转换为HTML

## 功能需求

- 读取一个.md文件
- 把标题（# ###）、段落、列表转换成对应HTML标签
- 输出.html文件

## 伪代码

在这里写你的伪代码：

```
你的伪代码：
    调用相应的库读取markdown文件，做判断确认文件存在以及内容不为空
    将标题对应<h1>、<h2>、<h3>等标签
    将段落对应<p>标签，如果遇到空行则使用换行<br>标签
    如果有链接，将其转换为<a>标签
    如果有图片，将其转换为<img>标签
    将列表对应<ul>、<ol>、<li>标签
    输出为.html文件
```
