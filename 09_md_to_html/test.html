日常开发中推荐使用 uv 管理 Python 环境，搭配 VS Code Pylint 做代码校验；~/.local/share/uv/python/3.11 是当前主力解释器。若出现 subprocess.Popen 启动报错，先执行 proxyreset 清空代理变量，再从干净终端 code . 重启编辑器，即可规避环境变量污染引发的 Linting 故障。
